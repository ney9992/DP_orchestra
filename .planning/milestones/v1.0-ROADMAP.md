# Milestone v1.0: Plant Simulation MVP

**Status:** ✅ SHIPPED 2026-05-17
**Phases:** 1-4
**Total Plans:** 9

## Overview

Первая рабочая версия DP_orchestra — Цифровой завод. Tauri v2-приложение (Rust + HTML/CSS/JS) с полной интеграцией Tecnomatix Plant Simulation: пользователь задаёт пути к файлам через панель настроек, запускает симуляцию, наблюдает лог в реальном времени и получает числовые результаты (загрузка линии, пропускная способность, время цикла) в UI.

## Milestone Summary

**Key Decisions:**

- Tauri v2 (Rust + HTML/CSS/JS) вместо PowerShell+WinForms: лучший UI через WebView2 при сохранении Windows-only развёртывания
- Запуск через `std::process::Command` (не COM API): меньше зависимостей от версий ПО
- Arguments format: `-f "{spp_path}" /E {method} --workdir "{work_dir}"` — SimTalk читает через getCommandLineArg
- work_dir (не директория exe) для results.txt: пользователь настраивает рабочий каталог
- inputSppPath/inputWorkDir: readonly + нативные диалоги (pick_file/pick_folder)
- Аккордеон: step-collapsed класс вместо pointer-events:none — locked шаги разворачиваемы
- history/ архив: work_dir/history/YYYY-MM-DD_HH-MM-SS.txt с заголовком метаданных
- Таймаут через mpsc::recv_timeout + taskkill PlantSimulation*.exe

**Issues Resolved:**

- Gap closure Phase 4.03: inputSppPath и inputWorkDir отсутствовали в HTML — закрыты планом 04-03
- UI accordion: locked шаги блокировали клики через pointer-events:none — исправлено введением step-collapsed
- CR-01 (advisory): set(id, val) guard оставлен как tech debt для v1.1

**Issues Deferred (Tech Debt):**
