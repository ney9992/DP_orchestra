## Открытие панели настроек

| Option | Description | Selected |
|--------|-------------|----------|
| Отдельное модальное окно (Form) | Кнопка «Настройки» в хедере — открывает отдельный Form. Простой паттерн для WinForms. | |
| Боковая панель в основном окне | Выезжает справа поверх карточек этапов, закрывается повторным кликом. | ✓ |
| Вкладка в основном окне (TabControl) | Добавляется вкладка «Настройки» рядом с основным видом. | |

**User's choice:** Боковая панель в основном окне
**Notes:** Пользователь предпочёл более современный вид без открытия нового окна.

## Как открывается панель

| Option | Description | Selected |
|--------|-------------|----------|
| Иконка «⚙» в хедере | Кнопка-иконка ⚙ в правом углу хедера. | ✓ |
| Кнопка «Настройки» в хедере | Полноценная кнопка с текстом рядом со статусом. | |
| Клик по любому пустому месту хедера | Любой клик в области заголовка открывает панель. | |

**User's choice:** Иконка «⚙» в хедере (рекомендация принята)

## Хранение настроек

| Option | Description | Selected |
|--------|-------------|----------|
| JSON-файл рядом с .exe | settings.json в $PSScriptRoot. ConvertTo/From-Json. | ✓ |
| JSON в AppData | $env:APPDATA\Bratsy_DP\settings.json. Работает без прав записи в папку .exe. | |
| Реестр Windows | HKCU\Software\Bratsy_DP. Встроен в Windows, но сложно переносить. | |

**User's choice:** JSON-файл рядом с .exe (рекомендация принята)

## Валидация пути

| Option | Description | Selected |
|--------|-------------|----------|
| При нажатии «Сохранить» | Проверка при попытке сохранить. Настройки НЕ сохраняются при ошибке. | ✓ |
| При потере фокуса из поля | Ошибка появляется, как только пользователь переместил фокус. | |

**User's choice:** При нажатии «Сохранить» (рекомендация принята)

## Отображение ошибки валидации

| Option | Description | Selected |
|--------|-------------|----------|
| Красная рамка TextBox + подсказка под полем | TextBox обводится красным бордером, под ним мелкий текст «Путь не найден». | ✓ |

**User's choice:** Красная рамка TextBox + подсказка под полем (inline validation)
