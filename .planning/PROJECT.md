# DP_orchestra — Цифровой завод

## What This Is

DP_orchestra — центральный оркестрационный слой цифрового завода: Windows-приложение (Tauri v2 — Rust backend + HTML/CSS/JS frontend), которое управляет передачей данных между промышленными системами (PDM, MES, BIM, имитационное моделирование, VR/3D, роботика). Реализует производственную цепочку **PDM → ТРП → MES → симуляция → физический завод**, позволяя команде проектировать завод, проверять гипотезы по продукту и сопровождать внедрение новых линий — без ручного перекладывания данных между программами.

Используется производственной командой (~несколько человек) для снижения рисков по CAPEX и ускорения старта реализации проекта.

## Core Value

По нажатию кнопки данные из всех подключённых систем проходят через пайплайн симуляции и возвращают экономические решения — оптимальная компоновка, загрузка, логистика, CAPEX.

## Архитектура системы

Центральный узел ("Цифровой завод") соединяет 6 контуров:

| Контур | ПО | Данные |
|--------|----|--------|
| Имитационного моделирования | Tecnomatix Plant Simulation, Anylogic | Компоновка, параметры станков, буферы |
| Детализированных роботических операций | Visual Components | Роботизированные ячейки |
| MES | (TBD) | Задание, статус, факт времени |
| VR и 3D-рендеров | Plant Simulation, Visual Components | Визуализация решений |
| Вспомогательного ПО и алгоритмов | Plant Simulation, LLM AI | Аналитика, поиск по проекту |
| BIM | Navisworks, Revit, Solidworks | Здание, мастерплан, оборудование |

Поток данных: `PDM (Vault)` → `ТРП (Excel)` → `MES` → `Цифровой завод` → `Физический завод`

## Context

- **Среда**: заводские ПК Windows, все интегрируемые системы (AutoCAD, Vault, PlantSim) работают на Windows
- **Существующий код**: `app/create_test.ps1` (423 строки) — полнофункциональный UI-прототип с симуляцией этапов
- **Метод интеграции**: запуск внешних скриптов и макросов (не COM API) — выбрано для простоты поддержки
- **Распространение**: один .exe файл, без установщика — удобно для заводской среды
- **Язык интерфейса**: русский

## Constraints

- **Tech stack**: Tauri v2 (Rust + HTML/CSS/JS) — выбрано для лучшего UI через WebView2 при сохранении Windows-only развёртывания
- **Platform**: Windows only — все интегрируемые системы требуют Windows
- **Integration**: запуск скриптов/макросов — не COM/API, проще поддерживать при обновлениях ПО
- **Distribution**: .exe без зависимостей — заводские ПК могут не иметь прав на установку пакетов
